# Sample Prompts — mlx-movie-director

Ready-to-use prompts for the moody Z-Image pipeline. Each prompt is designed for `--width 640 --height 960` portrait orientation.

## Usage

```bash
python/venv/bin/python python/mlx-movie-director/run.py \
  --prompt-file python/mlx-movie-director/app/prompts.md \
  --width 640 --height 960 --steps 9 --seed 42
```

## JK — 深夜自习室的"好学生"秘密补妆

Moody Photography，16-18岁日本清纯JK，透明感伪素颜妆，眼下淡粉色腮红，水光唇釉，冷白皮透出微微红晕，像刚跑完步。

深夜自习室的"好学生"秘密补妆
构图：半身略俯拍，女孩跪坐在房间地毯上，上半身趴在化妆台边缘，双手交叠垫着下巴，侧脸转向镜头，眼神有点慌张又有点倔强。

细节：深蓝色水手服上衣，领口第一颗扣子解开，白色领巾打了一个松松的蝴蝶结，里面穿白色蕾丝边吊带背心（只露出锁骨下一点点）。头发扎成双低麻花辫，辫尾系着两个小小的黑色兔尾巴毛球。右手捏着一支奶油粉口红，正要对镜补唇；左手看似随意地搭在腿上，其实指尖轻轻勾着格子裙下摆的边缘，隐隐露出大腿中段绑着的一条极细黑色吊袜带（没有链条，只是细丝带）。黑色过膝袜，袜口有白色蕾丝，勒出一小截肉感。嘴唇微张，舌尖轻轻抵着上颚，眼神迷蒙又干净。
背景：摊开的英文课本、写了一半的日记、一瓶波子汽水、半杯融化的哈根达斯，椅子背上挂着一件黑色的兔女郎发箍。
